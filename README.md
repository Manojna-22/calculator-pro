# CALCPRO MK-IV — Industrial Calculator

No frameworks. No dependencies. Open `index.html` and run.

## Run in VS Code
1. Install **Live Server** extension (Ritwick Dey)
2. Right-click `index.html` → **Open with Live Server**
3. Or just double-click `index.html` in Explorer

## Modes
| Mode | What it does |
|---|---|
| STD | Full standard calculator with memory |
| SCI | Trig, log, ln, powers, factorial, π, e, INV mode |
| PROG | HEX/DEC/OCT/BIN + bitwise AND/OR/XOR/NOT/LSH/RSH |

## Keyboard Shortcuts
| Key | Action |
|---|---|
| 0–9 | Digits |
| . | Decimal |
| + - * / | Operators |
| Enter / = | Equals |
| Escape | Clear (AC) |
| Backspace | Delete digit |
| Delete | Clear entry (CE) |
| % | Percent |
| N | Negate (±) |
| M | Memory Store |
| R | Memory Recall |
| A–F | Hex digits (PROG mode) |

## Features
- Operation tape (left panel) — full history, click to restore
- Unit converter (right panel) — Length / Weight / Temperature / Area
- Quick keys: π, e, √2, φ, 1K, 1M, 1B, 1%
- Last 5 results panel
- Memory: MC MR MS M+ M−
- Degree/Radian toggle
- INV mode for trig inverses (asin, acos, atan)
- CRT scanline effect + animated green display
